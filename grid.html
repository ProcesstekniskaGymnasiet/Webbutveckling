Design, struktur och uppbyggnad av webbplatser

Floats

Ursprungligen för flytande bilder i textblock har float egenskapen blivit ett av de vanligaste verktygen för att skapa flera kolumnlayouter på webbsidor. Syftet med floats är att lära dig hur man skapar drop caps, floated bilder där texten wraps runt och flera kolumnlayouter på webbsidor.

Floats låter dig lägga in blocknivåelement sida vid sida istället för ovanpå varandra. Det här är en stor sak. Det låter oss bygga alla slags layouter, inklusive sidofält, sidor med flera kolumner, galler och artiklar i tidskriftsformat med text som flyter runt en bild. Här börjar vi slutligen skapa riktiga webbsidor. MDN har en bra exempel. Och här en till från W3 School.

Flexbox

Float-baserade layouter har oftast ersatts med Flexbox på moderna webbplatser. Men det är inte att säga att Floats inte är värt att läsa. I över ett decennium fungerade flottor som grunden för de flesta webbplatser på Internet, vilket innebär att du definitivt kommer att stöta på dem någon gång i din karriär.

Layoutläget "Flexiboks" eller "Flexbox" erbjuder ett alternativ till Floats för att definiera det övergripande utseendet på en webbsida. Medan flottörer bara låter oss placera våra lådor horisontellt, ger flexbox oss fullständig kontroll över inriktningen, riktningen, ordningen och storleken på våra lådor.

Grid layout system

CSS Grid Layout är det mest kraftfulla layoutsystemet tillgängligt i CSS. Det är ett 2-dimensionellt system, vilket betyder att det kan hantera både kolumner och rader, till skillnad från flexbox som i stor utsträckning är ett 1-dimensionellt system. Du arbetar med Grid Layout genom att tillämpa CSS regler både på ett moderelement (som blir Grid Container) och till det här elementet barn (som blir Grid Items).
